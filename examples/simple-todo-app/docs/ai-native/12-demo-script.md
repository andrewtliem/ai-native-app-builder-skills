## Opening
Course Todo helps students track course-related tasks locally in the browser without creating an account.

## Problem Statement
Students often lose track of homework, readings, quizzes, and project tasks because information is scattered across many platforms.

## Demo Flow
1. Show empty task list.
2. Add task “Read Chapter 3” for course “MIS”.
3. Show validation by trying to add an empty task.
4. Mark a task complete.
5. Filter tasks by course and status.
6. Refresh browser and show tasks remain.

## Evidence to Mention
- FR-001 and FR-002 are shown by add task and validation demo.
- FR-005 and FR-006 are shown by filters.
- FR-007 is shown by refresh persistence.

## Limitations
- No cloud sync.
- No login.
- No collaboration.
- Automated tests are recommended but not included in this compact example.
