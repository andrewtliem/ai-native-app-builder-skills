# Starter Bundle

**Purpose:** Move from raw idea to clear intent and a student-friendly PRD.

**When to use:** Use in Week 1 or before students commit to a project idea.

## Included Skills / Practices

| Skill | Role |
|---|---|
| `01-grill-my-idea` | Challenge the raw idea before PRD or coding. |
| `02-idea-to-intent` | Capture users, problem, constraints, tradeoffs, and success criteria. |
| `03-intent-to-prd` | Create a PRD with user stories, requirements, and acceptance criteria. |

## Expected Student Artifacts

- `docs/ai-native/01-grill-my-idea.md`
- `docs/ai-native/02-intent-brief.md`
- `docs/ai-native/03-prd.md`

## Teaching Note

Students should run these skills one at a time, read the saved artifact, make a human decision, revise if needed, then continue. The bundle is a teaching path, not an autopilot.

## Repository Paths

Installable skills live under `skills/<skill-name>/SKILL.md`. This bundle is only a learning path; it does not duplicate the skill content.
